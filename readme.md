# Timeline Format
```
# Ext: .timeline

# For file global tags:
:tags ....

# Form One: Event
year       event country person* :desc .... :tags ... :wiki ....

# Form Two: Period:
year -> year event country person_surname* :tags ... :wiki .... :desc ....
```
